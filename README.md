**🖐️ FingerConnect - ASL Learning Web App**

**FingerConnect** is a beautifully designed, interactive web application that helps beginners learn the **American Sign Language (ASL) manual alphabet (A-Z)** through visual hand gestures and smooth animations.

**🌟 Features**

- 🎯 **Interactive A-Z Grid**
  - Displays all 26 ASL alphabet signs
  - Hover to reveal letter and details
- 🎨 **Modern UI Design**
  - Soft color palette (cream, amber, sage tones)
  - Smooth animations and transitions
  - Responsive layout for mobile and desktop
- 🧠 **Beginner-Friendly Learning**
  - Clear visual representation of hand signs
  - Encourages self-paced practice
- ⚡ **Lightweight & Fast**
  - Pure HTML, CSS, and JavaScript
  - No frameworks or dependencies
- ♿ **Accessibility Support**
  - Keyboard navigation (tabindex)
  - ARIA labels for screen readers

**🛠️ Tech Stack**

- **HTML5** - Structure
- **CSS3** - Styling & animations
- **Vanilla JavaScript** - Dynamic rendering of alphabet cards
- **SVG** - Hand gesture illustrations

**📸 How It Works**

- The app dynamically generates all alphabet cards using JavaScript.
- Each letter is mapped to a custom **SVG hand gesture**.
- On hover:
  - The hand fades
  - The letter is revealed
- Smooth animations enhance the learning experience.

**💡 Learning Tip (Built-in Feature)**

Start with just 5 letters a day. Practice in front of a mirror and try spelling your own name first!
